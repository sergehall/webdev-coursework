Project Overview

This submission contains all solutions for Tasks 6.17 to 8.7 in a single HTML file with clean user interaction, input validation, and HTML5 output.

------------------------------------------------------------

Submission Contents:

1. assignment3.html
- Entry point of the project.
- Contains the interface to run individual task scripts and test scripts.
- Uses modern HTML5 elements, <main>, <footer>, responsive layout, and styles.

2. assignment3.js
- Contains implementations of all task scripts:
  - compareTwoIntegers() – Task 6.17
  - calculateStats() – Task 6.18

  - calculateEvenOddProduct() – Task 8.7

3. testAssignment3.js
- Contains automated test logic for each task.
- Each test simulates input and outputs results using outputHTML().
