Note: This project uses JavaScript fetch() to dynamically load some files (like mod2-readme.html and mod-2-preview.html). To view everything properly, run the page on a local server using npx serve . or the Live Server extension in VS Code. Opening directly as file:// may prevent content from loading.

Files Included and Their Roles

1. assignment2.html

  • Main structure of the assignment page.
  • Includes:
    • A button to view tasks in a modal (View Full Task)
    • A button to preview the solution (Open Assignment Preview)
    • Modal windows for both tasks and preview content
    • JavaScript logic from mod-2-scripts.js
    • External styling from CSS files

2. mod-2-modal-tasks.css

  • Styles the task modal structure and appearance:
    • .modal-content, .tasks-header, .tasks
    • Buttons: .open-modal-btn, .close-button

3. mod-2-preview-btn.css

  • Styles the “Open Assignment Preview” button:
    • Background, padding, hover behavior

4. mod-2-preview.html

  • Contains solutions for Assignment – Module 2 in CS 80 – Internet Programming (Section 1239).
  • Used to preview completed tasks inside an iframe or shadow DOM.

5. mod-2-scripts.js

  • Implements interactive behavior for task modal, assignment preview, and dynamic footer.
  • Features include:
    • openModal(), closeModal() – toggle task description modal
    • togglePreview(), closePreviewModal() – control preview via Shadow DOM
    • Handles dynamic footer visibility based on scroll direction

6. assignment-header-readme-block.css

  • Provides styling for the assignment page header and task panel layout.
  • Also defines styles for the .readme-block section, including spacing, borders, font, and typography.

7. mod2-readme.html

  • Dynamically loaded into the page to display this README section.
  • Provides a structured overview of all project files and their functions.
