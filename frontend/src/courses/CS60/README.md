## Course Overview

This course introduces modern database concepts with an emphasis on the **relational database model**. Topics include:

- Database systems and architectures
- Data models and relational theory
- Entity-Relationship (ER) modeling
- Normalization of database tables
- Structured Query Language (SQL)
- Advanced SQL queries
- Database design methodologies
- Transaction management and concurrency control

Concepts are reinforced through **problem sets**, **chapter quizzes**, and **exams**, using a relational DBMS environment (Oracle SQL).
