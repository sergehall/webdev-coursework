## Course Overview

This course introduces core networking concepts and architectures, including network topologies, media and signaling, protocols and addressing, subnetting, internetworking, network services, and security. The course is aligned with **CompTIA Network+ (N10-008)**.

## Student Learning Outcomes

By the end of this course, students will be able to:

- Explain essential network components
- Select appropriate network topologies
- Compare network media
- Understand packets and protocols
- Evaluate network architectures
- Install and configure network applications
- Design custom network environments
