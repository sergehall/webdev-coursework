# Web Development & CS CourseworkPage Portfolio

This project serves as a personal portfolio and central hub for completed coursework across several core Computer Science and Web Development classes at Santa Monica College (SMC). Each section of this application represents hands-on work completed during online classes, including assignments, projects, and technical topics covered.

## Courses Included

- **CS 60** – Database Concepts and Applications
- **CS 70** – Network Fundamentals and Architecture
- **CS 73A** – Fundamentals of Computer Security
- **CS 79A** – Introduction to Cloud Computing
- **CS 79B** – Database Essentials in AWS
- **CS 80** – Internet Programming
- **CS 81** – JavaScript Programming
- **CS 83** – Server-Side Java Web Programming
- **CS 87A** – Python Programming

Each module demonstrates core skills, including:
- HTML, CSS, JavaScript
- React, TypeScript, Vite
- Security fundamentals
- Cloud and database technologies
- Server-side programming

## Purpose

This application is not for learning but rather to document and showcase my personal progress and deliverables from real courses. It's organized to reflect what I've mastered, and technologies are "unlocked" visually as I complete corresponding modules.
